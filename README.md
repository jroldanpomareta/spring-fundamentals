Repositorio con ejemplos prácticos de conceptos clave de Spring Framework.

## Contenido

* IoC (Inversión de Control)
* IoD (Inyección de Dependencias)
* Annotations
* Configuración de Spring

* ## Estructura

- ioc/
- iod/
- annotations/
- configuracion/

## Tecnologías

* Java
* Spring Framework
* Maven

## Objetivo

Aprender los fundamentos de Spring antes de desarrollar aplicaciones completas.

## Cómo ejecutar

Cada carpeta contiene un proyecto independiente.

Ejecutar con Maven:

mvn clean install
